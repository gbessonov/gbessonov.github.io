## gbessonov.github.io
This is a source repository for [the page on github.io](https://gbessonov.github.io/).
It uses [Next.js](https://nextjs.org/) framework.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
